# Δέσε Κορδόνια 🏃

Προσωπική ιστοσελίδα για το τρέξιμο — έμπνευση, πρακτικές συμβουλές και η προσωπική μου διαδρομή, από το πρώτο χιλιόμετρο μέχρι τον μαραθώνιο.

## Περιεχόμενα
- **Αρχική** — καλωσόρισμα, η ιστορία μου και γρήγορες συμβουλές
- **Ευεξία & Οφέλη** — πώς το τρέξιμο δυναμώνει σώμα και μυαλό
- **Επικοινωνία** — βρες με στο Instagram

## Τεχνολογία
Στατική ιστοσελίδα σε καθαρό **HTML & CSS** (χωρίς frameworks). Responsive για κινητό, tablet και υπολογιστή.

## Δομή
```
index.html            # ανακατεύθυνση στην αρχική
src/
  html/               # οι σελίδες
  css/style.css       # στυλ
images/               # logo, favicon, φωτογραφίες
```
